FIRST TIME SETUP (do this only once)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Copy this entire INTELLIGENT_OFFICE folder to your
   Desktop or C:\INTELLIGENT_OFFICE

2. Open the INTELLIGENT_OFFICE folder

3. Double-click:  02_INSTALL_HERE.bat

4. Wait for it to finish (5-15 minutes)
   → Say YES to any permission questions
   → Do NOT close the black window until it says "DONE"

5. Setup is complete!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
EVERY DAY: HOW TO START THE AI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Double-click:  start_all.bat
   (Wait 30-60 seconds for everything to load)

2. Your browser will open automatically to the AI chat

3. Start chatting! Type your questions.

4. When finished for the day:  double-click  stop_all.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
WHAT CAN THIS AI DO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💬 CHAT WITH AI
   → Just type questions in the browser chat
   → Ask anything! It knows a lot.

📄 CHAT WITH YOUR DOCUMENTS
   → Upload a PDF, Word file, or Excel in the chat
   → Ask questions about it
   → Example: "Summarize this report"
   → Example: "What are the key numbers in this file?"

📝 CREATE A WORD REPORT
   → Double-click:  generate_report.bat
   → Type your topic
   → AI writes a professional report for you!

📊 CREATE A POWERPOINT
   → Double-click:  generate_ppt.bat
   → Type your topic
   → AI creates slides automatically!

📈 CREATE AN EXCEL FILE
   → Double-click:  generate_excel.bat
   → Type your topic
   → AI creates a spreadsheet with charts!

🎙️ CONVERT AUDIO TO TEXT
   → Put your audio file in:  input_docs\audio\
   → Double-click:  transcribe_audio.bat
   → Text file appears in:  output_docs\reports\

🖼️ EXTRACT TEXT FROM IMAGE
   → Put image in:  input_docs\images\
   → Double-click:  ocr_image.bat
   → Text appears in:  output_docs\reports\
